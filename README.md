SecondBrain OS is an AI-powered personal knowledge system built for the **Missing Amps Winter Classic 2025**.

## Overview
SecondBrain OS allows users to upload their notes and documents and interact with them using natural language.
It uses semantic search and retrieval-based AI to generate context-aware responses.

## Key Features
- AI-powered semantic memory
- Natural language Q&A
- Retrieval-Augmented Generation (RAG) architecture
- Privacy-first, offline-friendly design
